# K-Means Clustering from Scratch

This repository contains a simple implementation of the K-Means clustering
algorithm written from scratch in Python using NumPy. The project also includes
visualization of clustering results using Matplotlib.

## 📌 Features
- Unsupervised learning algorithm
- Euclidean distance–based clustering
- Random centroid initialization
- Iterative centroid updates
- Convergence checking
- Cluster visualization

## 🛠️ Technologies Used
- Python
- NumPy
- Matplotlib
- scikit-learn (only for dataset generation)

## 📊 Dataset
Synthetic dataset generated using `make_blobs` from scikit-learn.
